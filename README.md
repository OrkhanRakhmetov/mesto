# Проект: Место

## Обзор
Сайт о меcтах и путешествиях, с возможностью редактирования профиля и добавления карточек с новыми местами.

## Используемые техники и технологии
- html
- CSS
- адаптивная верстка (Grid Layout, Flexbox, media queries)
- Java Script
- ООП
- Es6 Modules
- Webpack
